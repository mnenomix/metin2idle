# Feature 14: Pferde / Reitsystem

## Übersicht
Jeder Spieler kann ein Pferd besitzen das passive Stat-Boni gibt. Das Pferd wird durch Pferdemedaillen gelevelt (Drop in Affendungeons). Maximal 1 Pferd pro Charakter, max Level 30.

## Grundmechanik

### Pferd erhalten
- Beim NPC kaufbar (für Yang)
- 1 Pferd pro Charakter (nicht tauschbar/handelbar)

### Pferd leveln
- Durch **Pferdemedaillen** (Items)
- Pferdemedaillen droppen in **Affendungeons** (Hasun, Jungsun, Sangsun Dong)
- Pro Level-Up: X Medaillen nötig (steigend)

## Festgelegt
- **1 Pferd pro Charakter**
- **Max Level 30**
- **Leveln durch Pferdemedaillen** (Drop: Affendungeons)
- **Passive Stat-Boni** (VIT, INT, STR, DEX) - siehe Tabelle
- Level 30 hat deutlichen Bonus-Sprung (besonders INT: 22 → 61)
